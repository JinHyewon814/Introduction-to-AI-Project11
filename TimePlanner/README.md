# Time Planner

A Windows desktop planner built with Python and PySide6.

## Features

- Create tasks with names.
- Enter estimated duration in hours and minutes.
- Display a 6×24 timetable grid representing 24 hours in 10-minute cells.
- Save and load data from `data/planner.json`.

## Run

Install dependencies:

```bash
pip install PySide6
```

Run the app:

```bash
python main.py
```
